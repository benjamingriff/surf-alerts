## Codebase Patterns
- Frontend uses Next.js 14+ with App Router in packages/frontend
- shadcn/ui components with new-york style and slate base color
- Ocean-inspired color palette (blues/teals) in globals.css using OKLCH colors
- Inter font as default sans-serif via next/font/google
- Run `npm run dev` from packages/frontend to start dev server
- Run `npx tsc --noEmit` for typecheck

- Alerts context uses lazy state initialization to seed localStorage data without useEffect setState
- Signup form validation uses field-level error state with aria-invalid and text-destructive styles
